# JEIElement

## Import

~~~zenscript
import mods.randomtweaker.jei.JEIElement;
~~~

Elements rendered on the JEI, fluid slots, and item slot boxes are rendered from this

| ZenGetter                                                    | Return |
| :----------------------------------------------------------- | :----- |
| u                                                            | int    |
| v                                                            | int    |
| x                                                            | int    |
| y                                                            | int    |
| width                                                        | int    |
| height                                                       | int    |
| texture (Default : "randomtweaker:textures/gui/jei/jei_default.png") | string |
