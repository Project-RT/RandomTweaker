# JEIFontInfoElement

JEIFontInfoElement extends [JEIElement](JEIElement.md), so that all available `Getters` and `Setter`
of `JEIElement` can be used

## Import

```zenscript
import mods.randomtweaker.jei.JEIFontInfoElement;
```

| ZenGetter | Return |
| :-------- | :----- |
| info      | string |
| color     | int    |
