# JEILiquidSlot

JEILiquidSlot extends `JEISlot`, so that all available `Getters` of `JEISlot` can be used

## Import

~~~zenscript
import mods.randomtweaker.jei.JEILiquidSlot;
~~~

| ZenGetter / ZenSetter   | Return |
| :----------- | :------ |
| width        | int     |
| height       | int     |
| capacityMb   | int     |
| showCapacity | bool    |
