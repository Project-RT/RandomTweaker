# JEILiquidSlot

JEILiquidSlot extends [JEISlot](JEISlot.md), so that all available `Getters` and `Setter`
of `JEISlot` can be used

## Import

```zenscript
import mods.randomtweaker.jei.JEILiquidSlot;
```

## ZenGetterAndZenSetter

| ZenGetter / ZenSetter   | Return Type |
| :----------- | :------ |
| width        | int     |
| height       | int     |
| capacityMb   | int     |
| showCapacity | bool    |
