# JEISlot

Slots for storing data on JEI, currently only fluid and item slot are supported for rendering

## Import

```zenscript
import mods.randomtweaker.jei.JEISlot;
```

| ZenGetter / ZenSetter | Return Type |
| :-------- | :------ |
| input     | bool    |
| x         | int     |
| y         | int     |
| hasBase   | bool    |
