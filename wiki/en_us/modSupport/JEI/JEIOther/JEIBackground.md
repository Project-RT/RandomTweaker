# JEIBackground

JEI's background settings, you can set only the width and height

## Import

```zenscript
import mods.randomtweaker.jei.JEIBackground;
```

| ZenGetter / ZenSetter | Return Type |
| :----------- | :----- |
| resourceName | string |
| u            | int    |
| v            | int    |
| width        | int    |
| height       | int    |
