# JEILiquidSlot

继承 [JEISlot](JEISlot.md), 因此 `JEISlot` 的全部可用 `Getter` 和 `Setter` 都可被使用

## 导包

```zenscript
import mods.randomtweaker.jei.JEILiquidSlot;
```

## ZenGetterAndZenSetter

| ZenGetter / ZenSetter   | 返回类型 |
| :----------- | :------ |
| width        | int     |
| height       | int     |
| capacityMb   | int     |
| showCapacity | bool    |
