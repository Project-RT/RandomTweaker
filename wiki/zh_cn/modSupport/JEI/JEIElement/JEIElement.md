# JEIElement

## 导包

```zenscript
import mods.randomtweaker.jei.JEIElement;
```

渲染在 JEI 上的元素, 流体槽, 物品槽的框都是自此渲染

| ZenGetter / ZenSetter                                                    | 返回值 |
| :----------------------------------------------------------- | :----- |
| u                                                            | int    |
| v                                                            | int    |
| x                                                            | int    |
| y                                                            | int    |
| width                                                        | int    |
| height                                                        | int    |
| texture (默认 : "randomtweaker:textures/gui/jei/jei_default.png") | string |
